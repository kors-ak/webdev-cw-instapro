# Instapro | Photo Sharing Social Network

_Учебный проект по frontend-разработке_

Одностраничное веб-приложение социальной сети для публикации фотографий и взаимодействия с контентом пользователей.

## Демо

https://kors-ak.github.io/webdev-cw-instapro/

## Возможности

* Просмотр общей ленты публикаций.
* Авторизация и регистрация пользователей.
* Загрузка изображений через облачное хранилище.
* Создание новых публикаций с описанием.
* Просмотр профилей пользователей и их публикаций.
* Постановка и снятие лайков.
* Отображение количества лайков для каждой публикации.
* Сохранение данных авторизованного пользователя в LocalStorage.

## Технологии

* HTML5
* CSS3
* JavaScript (ES6 Modules)
* REST API
* Fetch API
* LocalStorage
* Git
* GitHub Pages

## Архитектура проекта

Проект построен на модульной архитектуре и разделен на независимые части:

```text
components/     — UI-компоненты приложения
api.js          — работа с серверным API
helpers.js      — вспомогательные функции
routes.js       — маршруты приложения
index.js        — точка входа и управление состоянием
```

Основное состояние приложения хранится централизованно и обновляется при переходах между страницами и выполнении API-запросов.

## Реализованные задачи

### Работа с API

* получение списка публикаций;
* получение публикаций конкретного пользователя;
* авторизация и регистрация;
* загрузка изображений;
* создание новых публикаций;
* управление лайками.

### Пользовательский интерфейс

* адаптивная лента публикаций;
* страница авторизации и регистрации;
* страница создания публикации;
* страница пользователя;
* индикатор загрузки данных.

### Безопасность данных

Для отображения пользовательского контента используется экранирование данных перед вставкой в DOM, что снижает риск XSS-уязвимостей.

## Что было улучшено дополнительно

* переработан внешний вид карточек публикаций;
* добавлены скругления и тени для улучшения визуального восприятия;
* реализовано собственное форматирование даты без подключения сторонних библиотек;
* уменьшено количество зависимостей проекта.
