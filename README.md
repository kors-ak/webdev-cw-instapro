# Instapro

MVP социальной сети для обмена фотографиями

## Реализованная функциональность

В рамках проекта успешно внедрены следующие функции:

- [x] интеграция вёрстки списка постов с бэкенд‑API;
- [x] страница добавления нового поста с формой загрузки изображения и описания;
- [x] персональная страница пользователя с отображением его постов;
- [x] функционал лайков (постановка/снятие лайка, отображение количества пользователей, поставивших лайк, и случайного имени из числа лайкнувших).

## Планирование и исполнение

### Первоначальная оценка

1. Интеграция верстки списка постов с API - 1 час.
2. Реализация страницы добавления поста - 2 часа.
3. Реализация страницы с постами конкретного пользователя - 2 часа.
4. Реализация функционала лайков - 2 часа.
5. Тестирование и доработка - 1 час.

   **Итого по плану:** 7 часов

### Фактически затраченное время

1. Интеграция верстки списка постов с API - 1 час 20 минут.
2. Реализация страницы добавления поста - 1 час 30 минут.
3. Реализация страницы с постами конкретного пользователя - 1 час (включая доработку дизайна).
4. Реализация функционала лайков - 2 часа 30 минут.
5. Тестирование и доработка - 1 час.

   **Итого фактически:** 7 часов 20 минут

## Дополнительные улучшения

В процессе разработки были внесены визуальные улучшения интерфейса:

- скруглены углы и добавлены тени для карточек постов;
- изменён цвет контейнера для фотографии на более светлый — для лучшей читаемости и эстетики.

Время на эти доработки учтено в рамках пункта «Реализация страницы с постами конкретного пользователя».

**Ключевое техническое решение, позволившее оптимизировать производительность:**

- собственная функция форматирования даты публикации вместо использования сторонних библиотек. Это снизило общий размер (вес) проекта и исключило лишние зависимости.
